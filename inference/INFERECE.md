YOLOv9 FastAPI Image Detection
=============================

Описание:
---------
Этот проект представляет собой локальный API на FastAPI для обработки изображений с использованием обученной модели YOLOv9.
Модель обрабатывает случайные изображения из папки `Data/images`, рисует bounding boxes с названиями классов и выводит результаты на веб-странице.

Структура проекта:
-----------------
project/
 ├─ main.py              # Основной код FastAPI
 ├─ Data/
 │   ├─ images/          # Папка с исходными изображениями
 │   ├─ output/          # Папка для сохранённых результатов с боксами
 │   └─ best.pt          # Весы обученной YOLOv9 модели
 └─ README.txt           # Этот файл

Установка:
----------
1. Убедитесь, что установлен Python 3.10+
2. Установите зависимости:

   pip install fastapi uvicorn pillow ultralytics

Запуск сервера:
---------------
1. Откройте терминал в папке проекта.
2. Запустите сервер:

   py -m uvicorn main:app --reload

3. Откройте браузер и перейдите по адресу:

   http://127.0.0.1:8000/

Использование:
--------------
- На веб-странице будут отображены **100 случайных изображений** из папки `Data/images` с:
  - Bounding boxes
  - Названиями классов, которые обнаружила модель
- Все обработанные изображения сохраняются в папку `Data/output`.
- Файлы в `Data/output` перезаписываются при повторной обработке, чтобы не накапливались старые результаты.

Структура кода:
---------------
- `main.py`:
  - `/` (GET) — веб-страница с изображениями и классами
  - Функция `process_images(max_images=100)` — обработка до 100 случайных изображений, рисование боксов и сохранение результата

Замечания:
-----------
- Для корректного отображения текста на изображениях используется шрифт Arial. Если шрифт не найден, используется стандартный.
- Если хотите видеть все изображения без ограничений, можно изменить параметр `max_images` в функции `process_images`.
